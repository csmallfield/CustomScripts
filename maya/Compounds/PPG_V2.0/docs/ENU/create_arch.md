# Create Arch
Experimental node.<br>
Creates an arch window for instancing.

### Radius
Radius for the arch.<br>
If this equals 0.5 \* "Width" the the arch is circular, otherwise pointed.

### Vertical Scale
Scale the arch section vertically.

### Width
The width of the window.

### Subdivs
Number of points between the beginning of the arc and the apex.

### Horizontal space
Space on each side of the window.

### Vertical Space
Space above and below the window.

### Bottom Height
The height of the lower (non-arch) part of the window.

### Extrude Depth
How deep to extrude the window in.
